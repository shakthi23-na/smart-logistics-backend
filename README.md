# Smart Logistics Backend

Backend service for Smart Logistics Transport System built using Spring Boot.

## Features
- REST APIs for logistics management
- CRUD operations
- H2 in-memory database
- CI/CD using GitHub Actions
- SonarCloud code quality analysis

## Tech Stack
- Java 17
- Spring Boot
- Maven
- H2 Database
- GitHub Actions
- SonarCloud

## DevOps
- CI pipeline triggered on push and pull request
- SonarCloud integrated for static code analysis
- Quality Gate passed
